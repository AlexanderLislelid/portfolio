# Portfolio

My personal portfolio website showcasing selected projects and skills as a Front-End Developer student.

## Features

- Overview of selected projects with links to live demos and GitHub repositories
- Dedicated section presenting the tools and technologies I regularly use
- Contact section allowing visitors to easily get in touch
- Responsive layout optimized for desktop and mobile devices
